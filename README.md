# Typing-study

타이핑 공부법을 이용해서 
블로그에 포스팅한 내용들을 바탕으로 
공부한 내용을 얼마나 알고있는지 기록

1. 복사 붙여넣기 x
2. 백지 or 제목만 작성하기
3. 주제에 대한 설명을 내가 알고있는대로, 이해한대로 작성하기
